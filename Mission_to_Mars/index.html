Mission to Mars

This is a homework assignment for the UM Data Analytics Bootcamp. In this assignment, a web application was built to scrape various websites for data related to the Mission to Mars. The information is displayed in this HTML page.

The tools used were: Python, Jupyter Notebook, BeautifulSoup, Pandas, Splinter, Flask and MongoDB
